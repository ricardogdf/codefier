<h1 align="center">Codefier</h1>

<p>
    Aplicação web com o intuito de codificar e decodificar textos de maneira eficiente e usual.
</p>

### Requisitos:

> - Deve funcionar apenas com letras minúsculas
> - Não devem ser utilizados letras com acentos nem caracteres especiais
> - Deve ser possível converter um texto para a versão criptografada e também retornar um texto criptografado para a versão original.
